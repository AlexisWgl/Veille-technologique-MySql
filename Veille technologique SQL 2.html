L'histoire de MySql

MySQL est un système de gestion de bases de données relationnelles. Il est distribué sous une double licence GPL et propriétaire. Sa 1ère version est sortie le 23 mai 1995 par la société MYSQLAB, son succès est dû à son caractère Open source. MySQL a été acheté le 16 janvier 2008 pour 1 milliard de dollars par Sun Microsystems. En 2009,Sun Microsystems a été acquis par Oracle Corporation, mettant entre les mains d'une même société les deux produits concurrents que sont Oracle Database et MySQL. MySQL est un serveur de base de données développé dans des performances élevées en lecture. MySQL fonctionne sur différents systèmes d'exploitation tels que Windows,Linux,Mac OS et supporte deux langages informatique : le langage de requête SQL et le SQL/PSM. Le langage de requête SQL est un langage informatique utilisé pour accéder aux données d'une base de données ou d'autres systèmes d'information. Il permet d'obtenir les données vérifiant certaines conditions, comme toutes les personnes qui habitent une ville donnée. Les données peuvent être triées, elles peuvent également être regroupées suivant les valeurs d'une donnée particulière. SQL / PSM ( SQL / modules persistants stockés) est une norme ISO qui définit principalement une extension de SQL avec un langage procédural à utiliser dans les procédures stockées .

MariaDB est un fork du système de gestion de bases de données SGBD MySQL. Son créateur, Michael Widenius (le plus souvent surnommé Monty) était également l’auteur de MySQL, et lorsque celui-ci fut racheté par Sun en janvier 2008 pour un milliard de dollars, Widenius travailla un temps pour la firme. Le 5 février 2009, il annonce sa démission et fondera peu après sa propre société, Monty Program AB. Le Finlandais souhaitait alors reprendre le développement d’un SGBD dans la lignée de MySQL, Sun ayant été racheté entre temps par Oracle. Widenius a toutefois adopté un modèle particulier pour ses activités : la totalité du code source est sous licence GPL mais tout code produit par les développeurs tiers doit être validé par son entreprise pour garder une cohérence. Entreprise qui, par ailleurs, appartient à ses employés, Widenius n’ayant lui-même aucune part. L’objectif de MariaDB est de progresser tout en gardant une compatibilité absolue avec MySQL. Cela comprend notamment une équivalence des bibliothèques, des API et des commandes. Une équivalence qui permet à MariaDB d’arriver comme remplaçant de MySQL sans nécessiter de trop grands travaux. Wikipedia a d’ailleurs franchi le pas en décembre 2012. D’autres sociétés ont également été séduites : Particuliers à Particuliers, Pierre et Vacances, Center Parc, Paybox ou encore la RTBF (Radio Télévision Belge Francophone).

 

Les licences

Deux types de licences sont proposées par MysQL:

- Licence Commerciale

Licence payante regroupant plusieurs éditions, MySql Standard, MySql Enterprise, MySql Cluster et MySql Embedded.

- Licence GPL (MySQL gratuit)

MySql Community Edition.

A savoir:

- MySql Community Edition est la version téléchargeable gratuite de MySQL, disponible sous licence GPL (General Public License) GNU.

- MySQL Standard Edition est l'offre de SGBDR d'entrée de gamme pour les applications de traitement de transactions.

- MySQL Enterprise Edition propose en plus des fonctionnalités avancées, des outils de gestion (dont OEM pour MySQL) et une assistance technique.

- MySQL Cluster Carrier Grade Edition est conçue pour le développement Web et Cloud.

MySQL Cluster est un SGBD open source adaptable, conçu pour fournir des performances et une disponibilité élevées à de nombreux types d'applications métier. Par exemple, MySQL Cluster pilote les bases d'abonnés des principaux fournisseurs de services de télécommunication et est utilisé au niveau mondial pour la détection des fraudes dans les transactions financières.

 

MySql Workbench

Workbench est un outil graphique puissant déstiné aux: BDA, développeurs et concepteurs. Il offre les outils nécessaires pour: Modéliser les données, développement SQL, la configuration du serveur de BD, sauvegarde etc. il est disponible sous Windows, Linux et Mac OS X

Les principales caractéristiques de MySql Workbench sont les suivantes:

General

  • Connexion à la base de données et gestion des instances
  • Actions dirigées par un assistant
  • Entièrement scriptable avec Python et Lua
  • Prise en charge des plugins personnalisés
  • Compatible avec MSAA (windows accessibility API)
  • Prise en charge des fonctionnalités de MySQL Entreprise

    • Éditeur SQL

  • Navigation, inspection et recherche des objets de schéma
  • Surligneur de syntaxe SQL et analyseur d'instructions
  • Complétion de code SQL et aide contextuelle
  • Ensembles des résultats multiples et modifiables
  • Visual EXPLAIN
  • Collections d'extraits de code SQL
  • Tunneling de connexion SSH
  • Support Unicode

    • La modélisation des données

  • Diagramme ER
  • Modélisation visuelle Drag'n'Drop
  • Synchronisation de schéma
  • Impresion de modèles

    • Administration des bases de données

  • Démarrer et arrêter des instances de base de données
  • Configuration de l'instance
  • Gestion de compte de base de données
  • Parcourir les variables d'instance
  • Navigation dans le fichier journal
  • Import/export de données

    • Suivi de la performance

  • Métriques du schéma de performances
  • Tableau de bord d'instance MySql
  • Statistiques de requête

    • Migration de base de données
