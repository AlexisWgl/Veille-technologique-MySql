Veille technologique

1ère version 23 mai 1995


MySQL a été acheté le 16 janvier 2008 pour 1 milliard de dollarspar Sun Microsystems. En 2009,Sun Microsystems a été acquis par Oracle Corporation, mettant entre les mains d'une même société les deux produits concurrents que sont Oracle Database et MySQL


MySQL est un serveur de base de données développé dans des performances élevées en lecture.

MySQL fonctionne sur différents systèmes d'exploitation tels que Windows,Linux,Mac OS et supporte deux langages informatique : le langage de requête SQL et le SQL/PSM.
