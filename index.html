L'histoire MySql

La 1ère version de MySql est sortie le 23 mai 1995 par la société MYSQLAB, son succès est dû à son caractère Open source. MySQL a été acheté le 16 janvier 2008 pour 1 milliard de dollars par Sun Microsystems. En 2009,Sun Microsystems a été acquis par Oracle Corporation, mettant entre les mains d'une même société les deux produits concurrents que sont Oracle Database et MySQL. MySQL est un serveur de base de données développé dans des performances élevées en lecture. MySQL fonctionne sur différents systèmes d'exploitation tels que Windows,Linux,Mac OS et supporte deux langages informatique : le langage de requête SQL et le SQL/PSM.
